# 📂 Dataset Information — CASIA-B Gait Database

---

## ⚠️ Why Is the Dataset Not Included in This Repository?

The **CASIA-B Gait Dataset** and its **preprocessed version** are extremely large in size:

| Data Type | Approximate Size | Description |
|---|---|---|
| **Raw Silhouette Images** | **~8 GB** | 124 subjects × 10 conditions × 11 angles × ~60–100 frames each |
| **Preprocessed `.pkl` Files** | **~4 GB** | Serialized NumPy arrays (64×64 grayscale frames per sequence) |
| **Trained Model Weights** | **~130 MB** | `gait_model.pth` + `gait_model_best.pth` |

> **Total: ~12+ GB** — far exceeds GitHub's file size and repository limits.
>
> GitHub enforces a **100 MB per-file limit** and recommends repositories stay **under 1 GB**.

### Steps to Download

1. **Visit** the official CASIA gait database page linked above.
2. **Fill out** the dataset request form with your academic/institutional information.
3. **Agree** to the terms and conditions for research use only.
4. **Download** the silhouette dataset (Dataset B — GaitDatasetB-silh.zip).
5. **Extract** into the `data/raw/` directory of this project following the structure below.

> 📌 **Note:** This dataset is available for **non-commercial, academic research purposes only**.
> Please cite the original paper if you use this dataset in your work.

## 📊 Dataset Statistics

| Property | Value |
|---|---|
| **Total Subjects** | 124 |
| **Walking Conditions** | 10 (6 normal + 2 bag + 2 coat) |
| **Viewing Angles** | 11 (0° to 180° in 18° steps) |
| **Total Sequences** | 124 × 10 × 11 = **13,640** |
| **Frame Resolution (Raw)** | 320 × 240 pixels |
| **Frame Resolution (Processed)** | 64 × 64 pixels |
| **Image Type** | Binary silhouette (grayscale) |

### Walking Conditions Explained

| Code | Condition | Description |
|---|---|---|
| `nm-01` to `nm-06` | **Normal** | Subject walks normally without any accessories |
| `bg-01`, `bg-02` | **Carrying Bag** | Subject walks while carrying a bag |
| `cl-01`, `cl-02` | **Wearing Coat** | Subject walks while wearing a long coat |

### Viewing Angles

```
0°   18°  36°  54°  72°  90°  108° 126° 144° 162° 180°
 ←────────────── Front to Back ──────────────→
```

- **0°** — Front-facing view
- **90°** — Side view (most discriminative for gait)
- **180°** — Back-facing view
